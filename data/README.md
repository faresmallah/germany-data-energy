# Data Guide

Place your dataset file here as `Germany_Sec4.xlsx`.

## Required Columns
- `year`
- `population`
- `oil_consumption` (TWh)
- `gas_consumption` (TWh)
- `oil_cons_change_pct` (% change YoY)
- `gas_cons_change_pct` (% change YoY)
